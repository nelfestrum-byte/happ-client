# Базовые настройки
1. Скачиваем клиент Happ:
	- **Android**: https://play.google.com/store/apps/details?id=com.happproxy
	- **IOS**: https://apps.apple.com/ru/app/happ-proxy-utility-plus/id6746188973
2. Открываем приложение, даем разрешение на получение уведомлений
3. Жмем шестеренку в верхнем левом и попадаем в настройки
4. Включаем последовательно :
	- Использовать фрагментирование
	- Включить шумы
	- Использовать Mux
	- Автозапуск приложения
5. Автозапуск попросит дать необходимое разрешение, соглашаемся и в списке приложений находим Happ и включаем
6. Выходим из настроек
7. Жмем `QR-Код` если конфиг в виде QR
8. Либо жмем `Из буфера` если конфиг в виде строки
9. В результате будет добавлены два конфига: 
	- `tcp-******-*******`  - для большинства случаев
	- `grpc-*****-*******` - лучше работает на мобильном интернете, лучше работают звонки в Телеграм, но "лагает" на плохом интернете
# Для продвинутых
При активации подписки в клиент автоматически загружаются стандартные правила маршрутизации, которые отправляют весь трафик по VPN за исключением российских IP адресов и некоторых популярных сервисов вроде банков. Если не работает какой то сервис можно попробовать принудительно включить или выключить для него VPN. Для этого в настройках можно найти разделы `Прокси для выбранных приложений` и `Маршрутизация`
## Прокси для выбранных приложений
Используется если нужно конкретное приложение принудительно заставить работать через VPN или наоборот работать без VPN, можно выбрать только один режим работы
Режимы:
1. ВЫКЛ - механизм отключен, все работает стандартно
2. ВКЛ - выбираем режим, отмечаем приложения из списка, отключаем и включаем VPN. В результате выбранные приложения будут ходить через VPN несмотря на стандартные настройки маршрутизации. Нужно использовать для нестандартные приложений, которым 
3. Обход - выбираем режим, отмечаем приложения из списка, отключаем и включаем VPN. В результате выбранные приложения будут работать БЕЗ VPN несмотря на стандартные настройки маршрутизации. Нужно включать для РФ сервисов типа госуслуг и банков, если они вдруг перестали работать
## Маршрутизация
Тонкая настройка на основе геосписков. По умолчанию маршрутизация включена предустановлен профиль `default`. Можно отредактировать списки зайдя в профиль-> Редактировать правила. `Глобальный прокси` - если включено, все идет через VPN кроме списка `Напрямую`, если выключено, все идет напрямую кроме списка `Прокси`. `Заблокировать` - список блокировки, можно использовать для предотвращения утечек. `Порядок` влияет на приоритет правил.
