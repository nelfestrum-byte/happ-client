# ⚙️ Базовые настройки
1. **Скачиваем клиент Happ:**
    - 📱 **Android**: [Google Play](https://play.google.com/store/apps/details?id=com.happproxy)
    - 🍎 **iOS**: [App Store](https://apps.apple.com/ru/app/happ-proxy-utility-plus/id6746188973)
2. Открываем приложение и разрешаем отправку **уведомлений** 🔔.
3. Нажимаем на иконку **шестерёнки** в верхнем левом углу, чтобы попасть в настройки.
4. Включаем последовательно следующие пункты:
    - ✅ Использовать фрагментирование
    - ✅ Включить шумы
    - ✅ Использовать Mux
    - ✅ Автозапуск приложения
5. При включении «Автозапуска» система попросит разрешение — соглашаемся, находим в списке приложений **Happ** и активируем переключатель 🚀.
6. Выходим из настроек.
7. Нажимаем `QR-код`, если ваша конфигурация представлена в виде изображения.
8. Или жмём `Из буфера`, если конфиг скопирован в виде строки текста 📋.
9. В результате будут добавлены два профиля:
    - `tcp-******-*******` — универсальный вариант для большинства случаев.
    - `grpc-*****-*******` — лучше подходит для мобильного интернета и звонков в Telegram, но может «лагать» при слабом сигнале 📶.

# 🛠 Для продвинутых пользователей
При активации подписки в клиент автоматически загружаются стандартные правила маршрутизации. Они отправляют весь заблокированный трафик через VPN, исключая российские IP-адреса и популярные локальные сервисы (например, банки).

> **Важно:** Приложения банков или Госуслуги всё равно могут видеть, что VPN включен, и выдавать предупреждение. Это не ошибка — фактически трафик к ним идёт напрямую, минуя прокси 🛡.

Если какой-то сервис всё же не работает, можно настроить исключения вручную в разделах `Прокси для выбранных приложений` и `Маршрутизация`.
## 📱 Прокси для выбранных приложений
Используется, если нужно принудительно заставить конкретное приложение работать через VPN или, наоборот, пустить его в обход. Можно выбрать только **один** активный режим:
1. **ВЫКЛ** — механизм отключен, всё работает по стандартным правилам.
2. **ВКЛ** — выбранные приложения всегда идут через VPN. Используйте это для сервисов, которые не определяются стандартными настройками 🌐.
3. **Обход** — выбранные приложения всегда работают **БЕЗ VPN**. Это спасение для банковских приложений и Госуслуг, если они вдруг перестали работать или если раздражает предупреждение о включенном VPN.
_После изменения настроек не забудьте выключить и снова включить VPN для применения изменений._

## 🚦 Маршрутизация — для тех, у кого «шаловливые ручки»
Тонкая настройка на основе гео-списков. По умолчанию включен профиль `default`. Редактировать списки можно в меню `Маршрутизация` -> `имя профиля` -> `Редактировать правила`.
- **Глобальный прокси**:
    - _Включено_ — весь трафик идет через VPN, кроме списка `Напрямую`.
    - _Выключено_ — весь трафик идет напрямую, кроме списка `Прокси`.
- **Заблокировать**: список для предотвращения утечек или блокировки рекламы/трекеров 🚫.
- **Порядок**: влияет на приоритет выполнения правил (верхние правила важнее нижних).
